Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito utilizando a <b> (não semântica)

Nesta frase, tenho um termo em destaque usando a <strong>, (possui semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico, usando a <i>

(Não semântica)

Nesta frase, temos um termo em ênfase usando a <em>

(semântica)

Texto marcado

Podemos criar também um texto marcado, usando a <mark> (Significado)

E no outro paragráfo, temos outro texto marcado no final. Utilizando a marcar especifica

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse paragráfo, usando a <big> e a <small>

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado, usando a <del>

Texto inserido

Podemos marcar um texto com inserido para dar mais ênfase e indicar que ele foi adicionado depois.

(Semântico)

Texto sublinhado

Existe também o sublinhado, usando a <u>

(Não semântica)

Para inserir coisas do tipo x20 + 3, usando a <sup>
